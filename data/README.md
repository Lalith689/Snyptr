# Sample Data

This folder contains sample images for testing SNYPTR.

## Usage

Place 2-3 sample shooting target images here for demonstration purposes. These images should show:
- Target shot with visible evidence of errors
- Good form examples
- Images in JPG or PNG format

## Example Images

- `good_shot.jpg` - Well-executed shot (reference baseline)
- `frontsight_dip_example.jpg` - Example of front sight dip error
- `overtight_grip_example.jpg` - Example of overtight grip error

## Notes

- Images with model files (.h5) are stored in Git LFS (Large File Storage)
- Sample images should be representative of actual shooting scenarios
- Minimum resolution: 224x224 pixels (model input size)
- Supported formats: JPG, PNG, JPEG
